Bem-Vindo ao IQuest

Aqui nenhuma pergunta fica sem resposta!

O IQuest é uma ferramenta de pesquisa onde os administradores podem cadastrar uma pesquisa e lançar para o seu grupo responder, IQuest de uma forma simples, fácil e ágil! ;)

IQuest acaba com as maneiras burocráticas de se realizar pesquisas, tanto para montar uma pesquisa quanto para responder, o usuário leva poucos minutos.

Para usar é bem simples, basta convidar as pessoas que irão participar daquele grupo especifíco (amigos, família, turma da faculdade, colegas de trabalho, etc.), se você for um Administrador conseguirá realizar pesquisas, caso seja membro responderá as pesquisas.
